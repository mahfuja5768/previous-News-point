Some important question

If you never want a variable to change, const is the keyword to use. If you want to reassign values: and you want the hoisting behavior, var is the keyword to use. if you don't want it, let is the keyword for you.

In non-restrict mode, regular functions allow us to use duplicate named parameters. But in strict mode, it is not allowed. Unlike regular functions, arrow functions do not allow duplicate parameters, whether in strict or non-strict mode. Duplicate parameters will cause a Syntax Error to be thrown.

Template literals provide an easy way to interpolate variables and expressions into strings. The method is called string interpolation.
